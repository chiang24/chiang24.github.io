分享一个z文件

z文件就是一个会记住曾经到过的目录的文件,有了它可以直接使用z. xxx(文件名)，到达你去过的目录

  1. 下载[z.sh]，放到一个不会删除的目录下
  2. 在 ~/.bashrc 里面加入z.sh的路径
    source /path/to/z.sh
  3. 运行 source ~/.bashrc
